Для ByRulesExchangeSpBehavior.
В методе Exchange перебираем список правил (ExchangeRules), в каждом правиле заданн (запрос, ответ, время на ответ, формат).
Также задается Resolution. Правила выполняется, если проходит условие в Resolution.

Resolution задавать в произвольной форме (не готовые имена св-в из uit)

Resolution= "{I}>20 || {U<100}"  //Правило выполнится если ток > 20 или напр. < 100

//Выбрать правила для отрисовки
var selectedRules = ExchangeRules.Where(rule => rule.CheckResolution<MyType>(data)).ToList();

CheckResolution(T data)
{
  через рефлексию ищу имя сво-ва с именем I и U.
  беру его значение и привожу к строке
  подставляю значение  I и U в выражене  "{I}>20 || {U<100}".
  вычиляю результат лог выраж.
}
